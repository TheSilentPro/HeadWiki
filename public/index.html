HeadDB

Thousands of heads at your fingertips!

Commands and Permissions

About
HeadDB gives you access to a database of thousands of heads!
With the running of a simple command, you can browse all of them.
 

Features

Database

A database with over 40.000 heads that automatically updates

Favorites Menu

Right-Click a head in the menu to add to favorites.

Local Heads

Heads from players that have joined the server.

Search Function

Easily Search for heads using /hdb search.

Give Function

Easily give heads to players using /hdb give.
